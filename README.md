# Zig Learning

<a href=""><img src="https://cdn.jsdelivr.net/gh/devicons/devicon@latest/icons/zig/zig-original.svg" width="250" height="250" class="cover" align="right"/></a>          

Aqui será documentado minha evolução com a linguagem de programação **Zig**, onde irei começar dos conceitos mais fundamentais aos mais avaçados da linguagem, e não apenas resolvendo exercícios propostos pelos materiais, como também desenvolvendo projetos.

O roadmap que planejei está além de apenas aprender **Zig**, também será utilizado outros materias para um domínio mais amplo da linguagem, e para reforçar conceitos um pouco enferrujados da minha parte.

O Roadmap consiste em cinco etapas, passando pelo aprendizado da linguagem, criação de algoritmos e desenvolvimeto de sistemas, sendo a última etapa um projeto final de um sistema operacional. Cada etapa é acompanhada de um livro que usarei de referência, e a etapa só estará concluída quando terminar o material (não necessariamente uma etapa adiante depende da anterior para iniciá-la).
